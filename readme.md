# Projet
    ---Araba Market
Un projet de marche en lign 

## Environement
* PHP 8.1.4
* Symfony 6
* symfony cli
* composer
* docker
* npm et nodejs

## Installation

```bash
composer install
npm install 
npm run dev
docker-compose up -d
symfony serve -d
```
